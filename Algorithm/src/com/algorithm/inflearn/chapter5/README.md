## Stack, Queue(자료구조)

### 1. 올바른 괄호

#### 예시 입력
```
(()(()))(()
```

#### 예시 출력
```
NO
```

<br/><br/>

### 2. 괄호문자제거

#### 예시 입력
```
(A(BC)D)EF(G(H)(IJ)K)LM(N)
```

#### 예시 출력
```
EFLM
```

<br/><br/>

### 3. 크레인 인형뽑기(카카오)

#### 예시 입력
```
5
0 0 0 0 0
0 0 1 0 3
0 2 5 0 1
4 2 4 4 2
3 5 1 3 1
8
1 5 3 5 1 2 1 4
```

#### 예시 출력
```
4
```

<br/><br/>

### 4. 후위식 연산(postfix)

#### 예시 입력
```
352+*9-
```

#### 예시 출력
```
12
```

<br/><br/>

### 5. 쇠막대기

#### 예시 입력
```
()(((()())(())()))(())
```

#### 예시 출력
```
17
```

<br/><br/>

### 6. 공주 구하기

#### 예시 입력
```
8 3
```

#### 예시 출력
```
7
```

<br/><br/>

### 7. 교육과정 설계

#### 예시 입력
```
CBA
CBDAGE
```

#### 예시 출력
```
YES
```

<br/><br/>

### 8. 응급실

#### 예시 입력
```
5 2
60 50 70 80 90
```

#### 예시 출력
```
3
```
