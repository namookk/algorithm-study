## Two Pointers

### 1. 단어중복없는 가장긴 문자열(Longest Substring Without Repeating Characters)

#### 예시 입력
```
s = "abcabcd"
```

#### 예시 출력
```
4
```

<br/><br/>

### 1. 최대 2 개의 고유 문자가있는 가장 긴 부분 문자열 (Longest Substring with At Most Two Distinct Characters)
#### 예시 입력
```
s = "eceba"
```

#### 예시 출력
```
3
```
